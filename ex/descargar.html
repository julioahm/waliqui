NOTA: Para reproducir el contenido necesitas instalar la extensión: Reproductor M3U8 - HLS + DASH Player

Instalación

Para instalar la extensión debes ingresar a Reproductor M3U8 - HLS + DASH Player o buscarla en la Chrome Web Store

Una vez instala la extensión recarge la página y disfrute del evento.
